================================================================
  🔧 KULLANIM KILAVUZU
================================================================

Normal başlatma:
    python app.py

Watchdog ile başlatma (çökünce auto-restart):
    python watchdog.py

Telegram aktivasyonu:
    1. config.json > "telegram" > "enabled": true
    2. .env dosyasına TELEGRAM_TOKEN ve TELEGRAM_CHAT_ID ekle
       (BotFather'dan bot oluştur, /start ile chat_id öğren)

API key kurulumu:
    .env dosyasını düzenle:
        AUSTRADE_API_KEY=...
        AUSTRADE_API_SECRET=...
